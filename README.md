# Algonive_Project_Task_Management

📌 Project Overview

The **Task Management System** is a web application designed to help small teams or individuals organize, track, and manage their daily tasks efficiently. It allows users to create tasks, assign priorities, update task status, and monitor progress in a simple and user-friendly interface.

This project focuses on improving productivity by providing a centralized platform for task tracking and workflow management.

---

🚀 Features

* **User Authentication** – Secure login and signup system
* **Create Tasks** – Add new tasks with title, description, priority, and deadline
* **Task Status Management** – Update task status (Pending, In Progress, Completed)
* **Task Dashboard** – View all tasks in an organized layout
* **Task Assignment** – Assign tasks to team members
* **Deadline Tracking** – Manage due dates to avoid missing important tasks
* **Responsive UI** – Clean and modern interface for better user experience

---

🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout design
* **JavaScript** – Dynamic functionality and task management logic
* **Local Storage / Database** – Storing user and task data
